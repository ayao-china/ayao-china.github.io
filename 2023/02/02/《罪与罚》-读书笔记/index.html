# 《罪与罚》读书笔记

# 前言

艰难地通读完《理想国》之后便接着翻开了书堆里的这本《罪与罚》，它的封面是一副抽象主义色彩的油画，也许这是在向想要打开这本书的读者表明，这本书不只有法律教育的意义，人们更应该挖掘深藏在其故事情节中的人性与哲学。

# 概述

故事前半段主要讲述了主角拉斯科尔尼科夫（罗季）策划杀死当铺老太婆之后，罗季在每一个场景的心理活动以及他的亲人朋友由于他的反常情绪而开始慢慢怀疑他是案件的凶手，尤其是波尔菲力（警察），然而让波尔菲力开始怀疑罗季的主要原因，竟然是罗季在大学时候发表的一篇关于犯罪的论文，这篇论文主要分析了罪犯在犯罪过程中的心理状态，论文最后得出结论：只要一个人对社会贡献足够大足够多，那么他是有权利做出触犯法律的事情而不受到惩罚，这样的理论让波尔菲力不得不怀疑，罗季的犯罪动机就是为了清除其实现伟大成就路途中的障碍。在戏剧性的背景下两人见面了，他们展开了一场激烈的辩论以论述这个看似荒唐逻辑诡异的议题。

# 摘抄笔记

这个思想无非是按照自然规律，人大体上分两种：一种是低等的（平等的），也就是所谓仅仅为繁殖同类而出力的材料；另一种是本来意义上的人，也就是有天赋或者有才华的人，能够在四周的人当中发表新的言论…… 第一种人永远是现在的主人，第二种人则是未来的主人。第一种人捍卫当前的世界，为它繁殖人口；第二种人推动世界前进，把它带到终极的目标。
—— 第三部第五章节选

  • 如果说前半段主人公的内心独白让我感到真实又可悲，如同自己就是那个杀人的凶手，每每读到他那草木皆兵的心理状态时，便如芒在背冷汗直冒。这段精彩的辩论却令我醍醐灌顶。罗季的理论填补了他的作案动机，也映射出了他极度自负的同时也有一颗柔软的心。

一个有良心的人，如果认识到自己的错误，就一定会痛苦，对他来说，这也就是苦刑之外的惩罚……. 如果他怜悯受害者，就让他痛苦吧，对宽广的理解力和深刻的心灵来说，痛苦和煎熬总是在所难免的。我认为真正伟大的人一定体验到世界上罪大的悲伤
—— 第三部第五章节选

  • 在反复挣扎中罗季遇见了索尼娅和斯维德力盖洛夫，作者安排这样一正一反的角色在主角身边，就是为了突出其复杂的心理矛盾，索尼娅竭力劝说拉斯科尔尼科夫自首，为其犯下的罪行赎罪，斯维德力盖洛夫则戏谑地看待罗季的行为，故意挑起争执，让罗季认为他即将要告发自己，还将自己谋杀发妻的事情对罗季和盘托出，这样奇怪的行径引起的罗季的警惕与愤怒，不过斯维德力盖洛夫就像是一面镜子，让罗季看清楚了自己的心理阴暗处。

“要知道，索尼娅，我杀害的不过是一只没用的，讨厌的，有害的虱子罢了”。“这只虱子可是个人啊”…….. 权利，只有那种敢于弯下腰，拾起他的人，才能得到。这只要有胆量就成，只求这一点。
仿佛突然间，有一团迷雾在他面前降下，把他包围在与世隔绝的状态里，使他无法冲出重围，弄的他很不好受。
—— 第六部第一章

  • 同时波尔菲力对罗季的层层逼问也压的他喘不过气来，每次与波尔菲力交锋，都让罗季如临大敌心力交瘁。

您知道他们有些人怎么看待受苦吗，这并不是为某个人受苦的问题，而干脆就是应当受苦，就是说人得受苦，如果是当官的弄的你受苦，那就更好
—— 第六部第一章

  • 波利菲尔对罗季说的这一段话是为了敲打罗季，那个承认犯罪的人是无辜的，他们虽然承认了罪行，但是只是为了完成自己的信仰，如同佛教的苦行僧。罗季最终自首被流放至西伯利亚，索尼娅也跟他一起去了，但是一开始罗季并不待见她，也许他是不想让索尼亚看见自己落魄受难的样子吧。可是罗季还是没有说服自己放弃可怕的犯罪理论和极端思想。

我的思想，” 他暗想，“怎么见得比其他那些从开天辟地以来就在世界上不断产生而且相互之间不断冲撞的思想愚蠢？怎么见得呢？只要用完全独立的、广阔的、摆脱世俗影响的目光来看问题，那么，当然，我的思想就显得根本谈不上那么…… 奇怪了。啊，你们这些喜欢否定一切的人，你们这些半吊子的圣人，为什么你们总是走到半路上就停下来了？
许多人类的恩人都不是从别人手里继承权力，而是自己夺得权力的，那就应该在他们刚开始迈步的时候就把他们处以极刑才对。然而，那些人偏偏把步子迈出去了，因此，他们就是对的，我呢，没有好，于是我就没有权利容许自己迈这种步子了
—— 尾声第二章节选

  • 也许是西伯利亚自由的空气和天使般善良的索尼娅让他慢慢放下思想的桎梏，重新接受新的生活，爱让一切都变得柔软，少年锋芒毕露的心最终也回归于人间烟火里的温暖

高陡的岸上望过去，眼前还展现出一片辽阔的郊野。从遥远的对岸隐约传来歌唱声。那边，草原一望无际，洒满阳光，上面有些看不大清的黑色斑点，那是些游牧者的帐篷。那边是自由的，在那边生活着的是另一种人，跟这边的犯人全然不同。
—— 尾声第二章节选

# 结语：

  • 在阅读过程中，除了罗季的成长及心理路程，有些配角也令我印象深刻，直率讲义气的拉祖米辛，纯洁勇敢的杜尼亚妹妹，内心阴暗性格怪诞的地主斯维德盖洛夫，他们的言行和对生活的态度为小说的故事情节走向铺垫了很多隐藏线索，也影响了处于案件漩涡中心的罗季。希望以后还会常翻常新发现更多有趣的情节，领会新的哲思。
