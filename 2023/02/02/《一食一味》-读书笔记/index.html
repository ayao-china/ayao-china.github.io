# 《一食一味》读书笔记

# 前言

俄国文学多描述苦难，赞颂信仰的力量，可是有时也不太有精力与头脑去感受书中的哲学，匆匆翻开这本写小食美味的散文，竟是教课书里令人垂涎的高邮鸭蛋，原来汪老不只写了一篇鸭蛋美食，书中还描绘了许多地域美食和各种小吃的做法，勤看可做菜谱，闲读则成散文。

# 摘抄笔记

家常酒菜

偶有客来，酒渴思饮。主人卷袖下厨，一面切葱姜，调作料，一面仍可陪客人聊天，显得从容不迫，若无其事，方有意思

  • 与朋友煮酒论道，闲谈小憩，乃人生一大乐事。之前与朋友闲谈，说工作之后对彼此最大的礼遇应该是在家亲自做饭宴请好友，现在看来属实不假。工作之后最缺的是时间，平时朝九晚六偶尔还要加班，与朋友约会已是难得，周末的时间都拿来打扫屋子，或学习或将时间投入兴趣爱好，在家研究烹饪技巧才是对现代年轻人的一种挑战。

# 结语：

  • 在阅读过程中，除了罗季的成长及心理路程，有些配角也令我印象深刻，直率讲义气的拉祖米辛，纯洁勇敢的杜尼亚妹妹，内心阴暗性格怪诞的地主斯维德盖洛夫，他们的言行和对生活的态度为小说的故事情节走向铺垫了很多隐藏线索，也影响了处于案件漩涡中心的罗季。希望以后还会常翻常新发现更多有趣的情节，领会新的哲思。
